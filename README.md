# VR Lockbox — интерактивная шкатулка в виртуальной реальности

VR-игра, в которой нужно найти красные шарики с цифрами-подсказками, ввести правильную комбинацию на шкатулке и получить приз.

## 🎮 Геймплей

1. **Найди 4 красных шарика** — они спрятаны в комнате. Лопни их, чтобы получить цифры-подсказки.
2. **Покрути барабаны** на шкатулке — выставь цифры в правильном порядке.
3. **Нажми кнопку «ОТКРЫТЬ»** — если код верный, шкатулка откроется.
4. **Забери приз** — внутри тебя ждёт кубок.

## 🛠️ Технологии

- **Unity 6** (6000.0.75f1)
- **XR Interaction Toolkit** — взаимодействие через лучи
- **Universal Render Pipeline (URP)** — графика
- **OpenXR** — поддержка VR-шлемов

## 🎮 Управление

| Действие | Управление |
|----------|------------|
| Навести луч | Движение контроллером |
| Нажать на объект | Trigger (курок) |
| Вращать барабан | Trigger + поворот контроллера |

## 🚀 Как запустить

1. Открой проект в **Unity 6000.0.75f1** или новее
2. Убедись, что установлены пакеты:
   - XR Interaction Toolkit
   - Universal RP
   - OpenXR
3. Открой сцену `Assets/Home/Scenes/MainScene.unity`
4. Нажми **Play**

## 🧩 Мини-игры в проекте

- 🎈 **Красные шарики** — лопни, получи цифру
- 🔐 **Шкатулка с кодом** — подбери комбинацию 3-5-1-7
- 📄 **Правила на бумажке** — возьми, прочитай, отпусти — вернётся на место
- 🏆 **Кубок в награду** — парит и вращается после открытия
